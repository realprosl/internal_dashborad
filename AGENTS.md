# Comandos de verificación

## Backend (Go)
- `go build ./...` - Compilar todo
- `go test ./...` - Ejecutar pruebas
- `go fmt ./...` - Formatear código
- `go vet ./...` - Análisis estático

## Frontend (Solid + TypeScript)
- `npm run build` - Construir para producción
- `npm run lint` - No configurado
- `npm run type-check` - No configurado

Se recomienda agregar scripts de lint y type-check en package.json.
